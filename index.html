E-Pole Fall Detection

This project was developed for the E-Yantra 2023 Innovation Challenge under the Disaster Management sector. It focuses on monitoring the orientation of electric poles using an MPU-based accelerometer sensor. When a pole begins to tilt or shows signs of falling, the system immediately triggers alerts to the concerned authorities. A responsive real-time dashboard was also built to visualize and track pole orientation remotely.
